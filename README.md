# Nestly
Aplikacija za jednostavan pronalazak nekretnina na osnovu ličnih preferencija korisnika i spajanje sa potencijalnim cimerima.

## Funkcionalnosti
* **Korisnički panel:** Pregled svih podataka.
* **Administrator modul:** Upravljanje korisnicima i bazom.
